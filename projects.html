Project Overview

How the Nigerian University Textbook Savings Guide was designed.

Project Idea

This project addresses the problem of textbook affordability for university students. Rather than simply listing books, the tool helps students compare expensive textbooks with free or low-cost alternatives and understand potential savings.

Features

  • Search by course code
  • Color-coded textbook cost labels
  • Free alternatives for selected courses
  • Nigerian context resources
  • Student tips for course preparation
  • Savings calculator for selected books

Methodology

The information was organized into a structured dataset with course codes, textbook recommendations, alternative resources, estimated savings, and study tips. The final version was converted into a web-friendly format using HTML, CSS, and JavaScript so that the project could be hosted directly on GitHub Pages.

Live Demo

The demo page contains the interactive version of the savings guide. Users can search, filter, and calculate their estimated textbook savings directly in the browser.
